# Netflix Data Analysis using Python

## 📌 Project Overview
This project analyzes the Netflix Movies and TV Shows dataset using Python, Pandas, and Matplotlib. The goal is to perform Exploratory Data Analysis (EDA) and discover useful insights from the data.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## 📂 Dataset
Netflix Movies and TV Shows Dataset from Kaggle.

## 📊 Analysis Performed
- Data Cleaning
- Missing Values Analysis
- Duplicate Check
- Movies vs TV Shows Analysis
- Top Countries
- Top Ratings
- Top Genres
- Release Year Analysis
- Content Added Over Time

## 📈 Visualizations
- Bar Chart
- Pie Chart
- Line Chart

## 🔍 Key Insights
- Netflix has more Movies than TV Shows.
- United States has the highest number of titles.
- Most content was added in 2019.
- Drama and International Movies are the most common genres.
